
# Amazon Sales Analysis (Python Project)

## Overview
This project analyzes Amazon sales data to understand customer behavior and product demand.  
The analysis shows that most customers are from Maharashtra, the majority of buyers are retailers
,orders are mainly fulfilled through Amazon, T-shirts are the highest-demand product, and M-size is 
the most preferred among customers.

## Libraries Used
- NumPy
- Pandas
- Matplotlib
- Seaborn

## Steps Performed
- Loaded the CSV file
- Cleaned missing and inconsistent data
- Checked dataset shape, columns, and info
- Analyzed sales by category, city, state, and order status
- Created simple visualizations (bar chart, line chart, count plot)

## Key Insights
- Maharashtra and Karnataka have the highest number of orders
- Top product categories include T-shirt, Shirt, and Kurta
- Most orders are Delivered or Shipped
- A notable number of orders were Cancelled

## Files Included
- Jupyter Notebook (.ipynb)
